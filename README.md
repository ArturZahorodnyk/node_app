# f1-app
## Link: [Formula One](https://forumula-one.herokuapp.com/home)

## Usage
### Production
**File:** `.env`

    PORT=<port>
    HOST=<host>
    NODE_ENV=production

**Execute:**
`npm run build`
`npm run start`

## Dev
**File:** `.env`

    PORT=?<port>
    HOST=?<host>
    NODE_ENV=dev

**Execute:**
`npm run serve`
